# First Screen UI Polish Walkthrough v1

작성 시각: 2026-06-14 20:21:02

## 구현 요약

첫 화면을 기능 확인용 테스트 화면에서 실제 게임 첫 화면에 가깝게 보이도록 HUD 정보 밀도와 배치를 정리했다.

이번 작업은 기능 추가가 아니라 표시 전용 UI polish다.

## 수정 파일

- `src/ui/Hud.ts`
- `src/scenes/GameScene.ts`
- `docs/UI_SYSTEM.md`
- `docs/PROJECT_STATUS.md`

## 첫 화면에서 줄인 정보

- 왼쪽 정보 패널 높이와 투명도를 줄였다.
- 공격/방어/EXP 문구에서 괄호와 긴 설명을 줄였다.
- 장비 보너스 문구를 한 줄 요약형으로 줄였다.
- 인벤토리 요약은 기존처럼 짧은 수량 정보만 유지했다.
- 긴 Battle Log는 계속 첫 화면에서 제외했다.

## 중앙 전투 영역 변경점

- 중앙 반투명 전투 박스를 완전히 제거하지 않고, 크기와 투명도를 낮췄다.
- 지역 배경과 오계장/몬스터가 더 먼저 보이도록 했다.
- 플레이어/몬스터 HP bar와 이름 표시 구조는 유지했다.
- 스킬 / 기본 타격 이펙트 표시 크기를 낮춰 전투 화면을 과도하게 덮지 않도록 했다.

## 우측 메뉴 / 전투 제어 변경점

- 우측 메뉴는 기본 펼침 상태를 유지한다.
- 스킬 / 장비 / 가방 / 퀘스트 버튼의 위치와 크기를 정리했다.
- 전투 제어는 기본 접힘 상태를 유지한다.
- 전투 제어 후보는 수동 / 오토 / x1.5 / x2를 유지한다.
- 실제 전투 속도나 수동 전투 로직은 연결하지 않았다.

## 하단 스킬 슬롯 / 장비 슬롯 변경점

- 스킬 슬롯 6칸은 유지했다.
- 스킬 슬롯 크기와 간격을 줄여 하단 UI가 중앙 전투 영역을 덜 침범하게 했다.
- 장착 장비 아이콘 슬롯 간격을 정리했다.
