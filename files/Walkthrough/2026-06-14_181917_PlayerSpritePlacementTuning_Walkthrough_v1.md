# 오계장 플레이어 스프라이트 배치 보정 완료 보고서 v1

작성 시각: 2026-06-14 18:19

## 구현 요약

오계장 최종 PNG가 실제 게임 화면에서 너무 작게 보일 수 있는 문제를 줄이기 위해 플레이어 스프라이트 원본의 불필요한 투명 여백을 잘라냈고, HUD 전투 화면과 상단 초상 표시 크기를 세로형 캐릭터 비율에 맞게 조정했다.

이번 작업은 표시 전용 보정이며 전투, 성장, 장비, 스킬, 보상, 스테이지 진행 로직은 변경하지 않았다.

## 수정 파일

```text
public/assets/characters/player/o_gyejang_idle.png
src/ui/Hud.ts
```

## 이미지 보정 내용

기존 PNG는 `1254x1254` 정사각형 이미지였고, 실제 캐릭터 주변에 투명 여백이 많았다.

보정 후 PNG 크기:

```text
760x1126
```

투명도 검증:

```text
PixelFormat: Format32bppArgb
HasAlpha: True
CornerAlpha: 0,0,0,0
```

## HUD 표시 보정

전투 화면의 오계장 이미지는 정사각형으로 찌그러지지 않도록 세로형 표시 크기로 조정했다.

```text
전투 화면 표시 크기: 198 x 294
상단 초상 표시 크기: 46 x 68
```
